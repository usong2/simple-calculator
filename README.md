<h1>Simple Calculator</h1>

간단하지만 사용자의 입력과 연산(처리) 그리고 출력을 수행하는 JavaScript 계산기입니다.

<br />

## 주요 구성 및 기능

+ 숫자 입력 버튼 : button 요소에 클릭 이벤트 핸들러를 적용해 입력을 판별
+ 사칙 연산 버튼 : 사칙 연산 부호를 입력
+ 초기화(clear) 버튼 : 현재 입력된 내용을 초기화
+ 결과(=) 버튼 : 계산 결과를 표시
+ 결과 창 : 버튼에서 입력한 내용을 표시하거나 계산 결과를 표시
